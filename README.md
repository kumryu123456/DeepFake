# 🎭 Deepfake Detection System

> AI Security Research Center, Soongsil University (2025.10 – 2025.12)

A deepfake video detection pipeline built during an AI Security research internship. Focused on data quality, model training, and improving detection accuracy through iterative labeling and consensus-based quality control.

---

## 📊 Key Results

| Metric | Before | After |
|--------|--------|-------|
| Label quality (consensus) | 78% | **95%** |
| Test accuracy | baseline | **92%+** |
| Dataset size | — | **6,170+ samples** |
| False-positive rate | 22% | **5%** |

## 🔍 What I Did

- **Data collection & labeling**: Collected and labeled 6,170+ deepfake/real video samples
- **Quality control**: Improved label quality from 78% → 95% via consensus-based review
- **Model training**: Fine-tuned PyTorch CNN models for binary deepfake classification
- **Evaluation**: Achieved 92%+ test accuracy; reduced false-positive rate from 22% → 5%
